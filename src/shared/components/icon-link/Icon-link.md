Icon Link (Cards with icon)

```
[Base]  .hw-icon-link
```


### Example
```html|span-2,plain,light
<a class="hw-icon-link" href="#">
  <svg class="hw-icon">
    <use xlink:href="#bud"></use>
  </svg>
  <h4>Card title</h4>
</a>
```
