
## Stickynav

The stickynav component.

Uses the `sticky` helper to create a sticky nav

```code
[Base]            .data-hw-stickynav
```

### Default behavior

```html
<nav class="hw-stickynav">
  <div class="hw-stickynav__inner" data-hw-sticky>
    <a href="#">Link 1</a>
    <a href="#">Link 2</a>
    <a href="#">Link 3</a>
    <a href="#">Link 4</a>
    <a href="#">Link 5</a>
  </div>
</nav>
```

Extra text to force page scrolling
