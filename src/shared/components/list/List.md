## List

Various list styles

```code
[Basic]                .hw-list

[Large list]           .hw-list--large
[Small list]           .hw-list--small
[Numbered]             .hw-list--numbered
[No bullets]           .hw-list--no-bullets

```



## Basic

```html|plain,light
<ul class="hw-list">
  <li>List item 1</li>
  <li>List item 2</li>
  <li>List item 3</li>
</ul>
```

## Large List
This is a list with lead style.
```html|plain,light
<ul class="hw-list hw-list--large">
  <li>List item 1</li>
  <li>List item 2</li>
  <li>List item 3</li>
</ul>
```
