
## Parallax

Adds parallax-effects to elements

```code
[Base]            [data-hw-parallax]

[Amount]          [data-hw-parallax-amount="2"] (optional, defaults to 1)
```

### Default behavior

```html
<div>
  <div class="
    hw-block
    hw-block--pt
    hw-block--pb
    hw-block--bg-primary
  ">
    <p data-hw-parallax data-hw-parallax-amount="2">Watch me stick!</p>
  </div>
</div>
```
