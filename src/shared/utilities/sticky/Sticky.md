
## Sticky

Makes wrapped contents stick to the top of the viewport.

The component left/right position will be based on the original width of the element/container

```code
[Base]            [data-hw-sticky]
```

### Default behavior

```html
<div data-hw-sticky>
  <div class="
    hw-block
    hw-block--pt
    hw-block--pb
    hw-block--bg-primary
  ">
    Watch me stick!
  </div>
</div>
```

Extra text to force page scrolling
