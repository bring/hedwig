## Buttons

Styling for buttons.

```code
[Base]           .hw-button

[Large]          .hw-button--large
[Small]          .hw-button--small
```

### Base

```html|span-4
  <button class="hw-button hw-button--large">
    Large
  </button>
  <button class="hw-button">
    Normal
  </button>
  <button class="hw-button hw-button--small">
    Small
  </button>
  <button class="hw-button hw-button--small" disabled>
    Disabled
  </button>
```

```code
[Base]           .hw-button
```
